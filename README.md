# Hwanyoung

## 프로젝트 관리 및 기술 스택

* 협업 및 버전 관리: 본 프로젝트는 Perforce(P4V)를 통해 소스 코드 및 대용량 에셋을 관리하며 협업을 진행하고 있습니다.

* 리팩토링 전략: 프로젝트 중도 합류 시점의 기존 블루프린트 중심 구조를 존중하되, 성능 최적화 및 유지보수 효율을 위해 단계적 C++ 리팩토링을 수행 중입니다.

* 코드 구성: 본 리포지토리는 제가 직접 C++로 컨버팅하거나 신규 설계한 모듈을 중심으로 구성되었습니다.

* 관전 포인트: 모든 기능이 C++로 전환된 상태는 아니나, 블루프린트와 C++의 확장성을 고려한 아키텍처 설계 방식에 중점을 두어 봐주시면 감사하겠습니다.

# 프로젝트 소개

[![Video Label](http://img.youtube.com/vi/W61dz_SI_PU/0.jpg)](https://youtu.be/W61dz_SI_PU)

📢 이 프로젝트는  **개발 중간에 참여한 프로젝트**입니다. 제가 첫 게임이 모든 것을 혼자 해보고 싶다는 생각이 들어 혼자 제작되었던 만큼, 이번에는 **협업을 통해 프로젝트를 완수해보고 싶다**는 생각으로 참여하여 제작하고 있습니다.

📢 개발 중간에 참여한 만큼, 기초 작업은 이미 진행되어있었고, 제가 참여했던 시점에 C++ 코드로 기초 작업을 한 후 **대부분의 작업을 블루프린트로 작업해놓았던 상태였기에, 현재는 블루프린트로 작업**하고 있습니다. 기능들이 구현이 되고 난 후에 C++로 변경이 가능한 기능들은 C++로 다시 구현할 예정입니다.
